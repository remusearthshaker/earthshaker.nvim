## ğŸ“¦ Installation (via `lazy.nvim`)

```lua
return {
  "remusearthshaker/earthshaker.nvim",
  lazy = false,
  priority = 1000,
}
```

Then in your LazyVim config:

```lua

  "LazyVim/LazyVim",
  opts = {
    colorscheme = "earthshaker",
  },
}
```

Or simply run within Neovim:

```vim
:colorscheme earthshaker
```
